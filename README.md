# Image Assets

This repository contains image assets organized by region/country for CDN delivery via jsDelivr.

## Folders

- `ECO-ID_2025-11-18_20_13/` - Indonesia images
- `ECO-TH_2025-11-18_20_13/` - Thailand images  
- `MY-EN_2025-11-18_20_13/` - Malaysia images
- `SG_2025-11-23_11_35/` - Singapore images
- `Vietnam_2025-11-18_20_13/` - Vietnam images

## CDN Usage

Access images via jsDelivr CDN:
```
https://cdn.jsdelivr.net/gh/Jack-1011/image-assets@main/FOLDER_NAME/IMAGE_NAME
```

Examples:
```
https://cdn.jsdelivr.net/gh/Jack-1011/image-assets@main/ECO-ID_2025-11-18_20_13/frame-12_3x_iujcri.png
https://cdn.jsdelivr.net/gh/Jack-1011/image-assets@main/ECO-TH_2025-11-18_20_13/green_programme-shared_post_th_1_mpmjev.jpg
https://cdn.jsdelivr.net/gh/Jack-1011/image-assets@main/MY-EN_2025-11-18_20_13/greenline-1_pkapcm.svg
https://cdn.jsdelivr.net/gh/Jack-1011/image-assets@main/SG_2025-11-23_11_35/animation1.svg
https://cdn.jsdelivr.net/gh/Jack-1011/image-assets@main/Vietnam_2025-11-18_20_13/Group97_rpif7s.svg
```
